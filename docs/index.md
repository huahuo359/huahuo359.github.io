## 课程笔记

### 本地运行
- 终端运行:
  
  切换到 `mkdocs/my-project/`目录下

  `mkdocs serve ` 进入浏览器进行本地的访问 

  `mkdocs gh-deploy` 部署在github page 上
  
  `ctrl`+`C` 可以正确退出 

- 核心内容:
  
  主要以一些参考的学习链接/一些备忘性质的重要内容/期末复习为主,由于目前对于电脑记录笔记并不太习惯,因此其中的个人笔记内容不是很多.

  笔记本由markdown进行即可.
